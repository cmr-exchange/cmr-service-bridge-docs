## Running the Tests [&#x219F;](#contents)

To run just the unit tests, use this command:

```
$ lein ltest :unit
```

Similarly, for just the integration tests:

```
$ lein ltest :integration
```

The default behaviour of `lein ltest` runs all tests types.
